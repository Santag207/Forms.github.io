Registro de
Muestreo de Aves

Recolecta datos biológicos en campo, adjunta fotografías y genera reportes LaTeX listos para publicación — sin conexión a internet.

📡 Offline 📱 Multiplataforma 📄 LaTeX / PDF
● Sin conexión

Cómo usar AveSampler

  1. 01
    Selecciona la plantilla

    Elige el tipo de reporte: estándar, biodiversidad o registro extendido. Ingresa el nombre del proyecto y la fecha.

  2. 02
    Completa los datos de muestreo

    Registra ubicación, coordenadas, método de muestreo, condiciones climáticas y las especies observadas.

  3. 03
    Agrega fotografías

    Adjunta imágenes desde tu galería o toma fotos directamente con la cámara. Formatos JPG y PNG.

  4. 04
    Genera el reporte

    El sistema clona la plantilla LaTeX, inserta los datos y genera un archivo .tex + .pdf comprimidos en un ZIP listo para descargar.

Funciona sin internet. Todos los datos se guardan localmente. La compilación a PDF requiere conexión si no tienes LaTeX instalado.
